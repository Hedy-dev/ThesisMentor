## 📋 О Проекте

**Demo Checker** — это интерактивный Jupyter-ноутбук, разработанный для образовательных целей. Система помогает студентам и преподавателям оценивать качество академических и бизнес-текстов по следующим критериям:

### ✅ Основные возможности

| Функция | Описание | Технология |
|---------|----------|------------|
| 🔍 **Структурный анализ** | Проверка наличия обязательных разделов: проблема, решение, гипотезы, метрики | Regex + RuBERT |
| 🔗 **Семантическая связность** | Оценка соответствия введения и практической части через косинусное сходство эмбеддингов | RuBERT + Mean Pooling |
| 🧠 **NLI-проверка** | Анализ логического следования между утверждениями | RuBERT-NLI (3-way classification) |
| 💡 **Генерация рекомендаций** | Персонализированные советы по исправлению ошибок | Шаблонная система + контекстная адаптация |
| 📊 **Классификация ошибок** | Разделение замечаний по уровням критичности | Rule-based + ML |

### 🎯 Для кого этот проект?

- 👨‍🎓 **Студенты** — проверка черновиков ВКР, курсовых и проектных работ перед сдачей
- 👩‍🏫 **Преподаватели** — демонстрационный материал для курсов по NLP, анализу данных и машинному обучению
- 🔬 **Исследователи** — пример интеграции предобученных трансформеров в прикладную задачу

## 🚀 Быстрый старт

### Вариант 1: Запуск в Google Colab (рекомендуется)

1. Нажмите кнопку **"Open In Colab"** в шапке этого файла
2. Дождитесь загрузки окружения
3. Последовательно запускайте ячейки кода (`Shift + Enter`)
4. Первый запуск займёт **5–10 минут** (загрузка моделей ~1.4 ГБ)

> 💡 **Совет**: Для ускорения работы в Colab выберите `Runtime → Change runtime type → GPU`

### Вариант 2: Локальный запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Cooper1952/my_demo_scenario.git
cd my_demo_scenario

# 2. Создайте виртуальное окружение (рекомендуется)
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите Jupyter Notebook
jupyter notebook Demo_checker.ipynb

Зависимости
Создайте файл requirements.txt со следующим содержимым:
